**npm i -g edu_node_cli**

##Использование

datetime <команда> <опции>

##Команды:

datetime current        Выводит текущую дату

datetime add [options]  Прибавляет заданные день/месяц/год к текущей дате

datetime sub [options]  Отнимает заданные день/месяц/год от текущей даты

## Опции:

-h, --help     Помощь [булевый тип]

--version  Показать номер версии [булевый тип]

-y, --year     Год [булевый/числовой тип]

-m, --month    Месяц [булевый/числовой тип]

-d, --date     Дата в календарном месяце булевый/числовой тип]
